# Infinite Server26 - SwiftUI Desktop Application

macOS desktop application for managing and monitoring Infinite Server26 Fortress.

## Requirements

- macOS 13.0 or later
- Xcode 14.0 or later
- Swift 5.0 or later

## Building

```bash
xcodebuild -project InfiniteServer26.xcodeproj \
  -scheme InfiniteServer26 \
  -configuration Release \
  clean build
```

## Features

- Real-time system monitoring
- AI systems control panel
- Security dashboard
- Blockchain visualization
- Container management
- System settings
